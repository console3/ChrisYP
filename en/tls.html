üîß POST Parameters (JSON Format)

Parameter Type Description Required
url String Request URL address ‚úÖ
method String Request method, get/post ‚ùå
headers String/Object Request headers, can be string or object, defaults to { 'User-Agent': 'random ua' } if not provided ‚ùå
cookies.value String/Object Cookies for request, can be string or object, { '1': '2', '3': '4' } / '1=2; 3=4' ‚ùå
cookies.uri String Domain for cookie setting, use url ‚ùå
proxy String Proxy address, supports protocols: http/https/socks5, format: ip:port or user:pass@ip:port ‚ùå
data String/Object POST request body, can be string or object, { '1': '2', '3': '4' } / '1=2&3=4' ‚ùå
json String/Object POST JSON data, example { '1': '2', '3': '4' } ‚ùå
timeout Number Request timeout (seconds), default 15 seconds ‚ùå
http2 Boolean Whether to use HTTP/2 protocol, default false ‚ùå
redirect Boolean Whether to follow redirects, default true ‚ùå
ja3 String Custom JA3 fingerprint, defaults to latest Chrome random fingerprint if not provided ‚ùå
